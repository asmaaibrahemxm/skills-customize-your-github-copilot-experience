
# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a classic Hangman game in Python using strings, loops, conditionals, and random word selection. You will practice managing game state and handling user input to create a playable word-guessing experience.

## 📝 Tasks

### 🛠️ Set Up the Word-Guessing Game

#### Description
Create the core Hangman game loop and choose a secret word from a predefined list.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses from the player
- Show the current progress in `_ _ _` format
- Prevent the game from ending until the word is guessed or attempts run out

### 🛠️ Track Game Results

#### Description
Add logic to manage incorrect guesses and display the final outcome.

#### Requirements
Completed program should:

- Track how many incorrect guesses remain
- End the game when the word is guessed or attempts are exhausted
- Display a win message when the player guesses the word
- Display a lose message when the player runs out of attempts
